# 🌍 Tour & Travel Booking Website

A full-stack **Tour & Travel Booking Website** that allows users to explore travel packages, register for tours, and submit feedback.
The project demonstrates backend development using **Node.js, Express, and MongoDB** along with a responsive frontend built using **HTML, CSS, and JavaScript**.

---

## ✨ Features

* 🌍 Browse available tour packages
* 👤 User registration system
* 🧳 Book travel packages
* 💬 Submit feedback
* 🗄 Store user data in MongoDB database
* 📱 Responsive travel website UI

## ⚙️ Installation

1️⃣ Clone the repository

git clone https://github.com/yourusername/tour-travel-booking.git

2️⃣ Navigate to backend folder

cd backend

3️⃣ Install dependencies

npm install

4️⃣ Start the server

node server.js

5️⃣ Open frontend

Open **index.html** in your browser.

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder.

PORT=5000
MONGO_URI=your_mongodb_connection_string
